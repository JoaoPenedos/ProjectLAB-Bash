Explicação passo-a-passo de como colocar a funcionar a script em diferentes máquinas (linux)
Ter em atenção as notas que se encontram no final do ficheiro

1ºPasso - Para correr a script deve abrir o terminal e se colocar no caminho onde se encontra a script(*Nota-2*), de modo a 
não criar erros.

2ºPasso - Para correr a script coloque o seguinte comando no terminal ". /scriptSH.sh".

2.5º - Comece por dar unzip a pasta Videoteca que se encontra na mesma pasta da script (pode fazer isso no menu depois de correr a script)

3ºPasso - De seguida deverá escolher as opcões no Menu baseado nas opções que deseja executar.

*Nota-2*: exemplo, se a script encontra-se em "~/Trabalho2/scriptSH.sh" deverá se colocar em "/Trabalho2"
